PHP 301 Redirect

Mar 18th, 2011

Just using the standard:

header(“Location: http://www.google.co.uk”);

uses a 302 redirect.

However, doing:

header("HTTP/1.1 301 Moved Permanently");
header("Location: http://www.google.co.uk");

Does a proper 301 permanent redirect.
